Чтобы запустить расширение, необходимо:
1) Внутри папки server: Выполнить npm install
2) Добавить в папку models модель с расширением .gguf
3) Заменить modelPath: "models/Qwen3-1.7B-Q8_0.gguf" в файле index.js на вашу модель. 
4) С помощью node js версии 21 или выше выполнить node index.
5) Загрузите расширение с помощью edge://extensions/   →   загрузить распакованное
